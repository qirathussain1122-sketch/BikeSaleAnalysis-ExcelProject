#Bike Sales Analysis: Excel Project

##Project Overview

This is a **simple Excel data analysis project** focused on cleaning, organizing, and analyzing bike sales data.

## Data Cleaning Steps

The following steps were performed to prepare the dataset for analysis:

* Removed **duplicate records** to ensure data accuracy
* Replaced **incomplete or abbreviated letters** with full, meaningful words
* Converted **currency values to numeric format** for easier calculations
* Created a new **Age Bracket** column to group customers by age

---

##Data Analysis
After cleaning the data, analysis was performed using Excel’s built-in tools:
* Created **pivot tables** to summarize bike sales by different categories
* Built **charts** from pivot tables to visualize trends and patterns
## Tools Used
* Microsoft Excel
* Pivot Tables
* Charts
